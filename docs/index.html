CLD CTRL

Mission control for Claude Code. Switch projects, resume sessions, and launch tasks — all from a single hotkey.

Get Started View on GitHub
$ npm i -g cldctrl

Ctrl+Up → pick project → Claude Code in under 2 seconds

Features

Everything you need to manage Claude Code projects

>_

Instant Popup

Ctrl+Up opens a mini TUI in ~300ms. Pick a project, pick an action, go. Pre-warmed for instant response.

▶

Project Launcher

Open Explorer, VS Code, and Claude Code in one action. Resume your last conversation or start fresh.

☷

Session Management

Browse session history with token counts, tool usage, and model info. Resume any session instantly.

☋

Git Status at a Glance

Branch name, uncommitted changes, and unpushed commits shown inline for every project.

⚠

GitHub Issues

See open issues per project. Launch Claude Code with "fix issue" prompts directly from the dashboard.

⚙

Auto-Discovery

Finds projects from ~/.claude/projects automatically. Also scans for project indicators like .git, package.json, and CLAUDE.md.

■

Usage Tracking

Daily token and message counts with rate limit monitoring. Calendar heatmap shows activity over time.

/

Filter & Search

Type / to fuzzy-filter projects in any view. Keyboard-driven navigation throughout.

▣

Two Modes

Mini popup for quick launches, full dashboard for project management. Both modes, one tool.

Quick start

Up and running in 60 seconds

1

Install

npm i -g cldctrl
2

Launch the dashboard

Run cldctrl (or the shorter aliases cld / cc) to open the full TUI.

cldctrl          # full dashboard
cldctrl --mini   # mini popup
3

Set up the hotkey (Windows)

Register Ctrl+Up as a global hotkey for the mini popup.

cldctrl setup
